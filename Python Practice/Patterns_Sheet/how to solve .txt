Step 1 (Always)

Forget numbers.

Just draw the shape.

Example:

*****
 ****
  ***
   **
    *

Ask only:

Where are the spaces?
Where are the characters?

Nothing else.

Step 2

Ask:

What decides the printed value?

There are only four possibilities.

Case 1

Depends only on i

Example

55555
4444
333

Value = i or n-i+1

Case 2

Depends only on j

Example

12345
1234
123

Value = j

Case 3

Depends on i and j

Example

123
234
345

Value = something like

i+j-1
Case 4

Needs a counter

Example

A
BC
DEF

No simple formula.

Use

p += 1

Done.

Step 3

Only if you find yourself writing

c += 1

ask:

Can I replace this by a formula?

Not before.

Why?

Because formulas are difficult.

Coordinates are easy.

Example.

Today's pattern.

ABCDE
 A__D
  A_C

Did you need a formula?

No.

You only needed coordinates.

Another example.

12345
 1234
  123

Again.

No algebra.

Only coordinates.
